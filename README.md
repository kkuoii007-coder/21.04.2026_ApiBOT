# Telegram Bot для поиска ценных бумаг на Московской бирже

Простой Telegram бот, который ищет акции и облигации конкретного эмитента на Московской бирже.

## Установка

1. Создайте виртуальное окружение:
   ```bash
   python3 -m venv venv
   ```

2. Активируйте виртуальное окружение:
   - В Linux/Mac:
     ```bash
     source venv/bin/activate
     ```
   - В Windows:
     ```bash
     venv\Scripts\activate
     ```

3. Установите зависимости:
   ```bash
   pip install aiogram requests
   ```

## Настройка

1. В файле `config.py` укажите токен вашего Telegram бота:
   ```python
   TOKEN = 'ваш_токен_бота'
   ```

## Запуск

Запустите бота командой:
```bash
python main.py
```

## Использование

- Команда `/start` - приветствие и инструкция
- Отправьте текстовое сообщение с названием эмитента (тикер, код бумаги или официальное наименование)
  
Примеры запросов:
- SBER
- GAZP
- Лукойл
- Сбербанк

Бот вернет отдельно найденные акции и облигации.
